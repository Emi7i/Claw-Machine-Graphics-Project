# Claw Machine Graphics Project

A 3D claw machine simulation with ReactPhysics3D physics engine and OpenGL rendering.

## Dependencies

Install dependencies via vcpkg:

```bash
vcpkg install reactphysics3d
vcpkg install stb
vcpkg install glad
vcpkg install glfw3
vcpkg install glm
vcpkg install assimp
```

## Setup

1. Clone this repository
2. Install dependencies with vcpkg (see above)
3. Add your own model files to the `res/` folder:
   - `claw_machine.obj`
   - `claw_machine.mtl`
   - `ground.obj`
   - `ground.mtl`
4. Open `Sablon.vcxproj` in Visual Studio
5. Build and run
